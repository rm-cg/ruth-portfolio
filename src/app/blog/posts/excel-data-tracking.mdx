## Advanced Formulas for Data Tracking

Tracking complex records is easily automated with advanced lookup functions to prevent human error and ensure data consistency.

<CodeBlock
    marginBottom="16"
    highlight="4"
    codes={[
      {
        code:
`=IFERROR(
  INDEX(MasterList[Status],
  MATCH([@ID_Number], MasterList[ID_Number], 0)),
  "Pending"
)`,
        language: "plaintext",
        label: "data_tracker.xlsx"
      }
    ]}
/>

The above declaration automatically checks the master database and flags mismatched IDs. You can use a static text warning by replacing the `"Pending"` value with a different string in the formula.
