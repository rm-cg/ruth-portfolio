## Workflow Automation

There's a pre-configured automation setup that you can modify in the `trelloBoard` object. Set workflow elements such as triggers, actions, due dates, and configure their appearance to optimize your team's workflow efficiency.

<CodeBlock
    codeHeight={24}
    marginBottom="16"
    codes={[
  {
    code:
`const trelloBoard = {
    action: 'https://api.trello.com/1/cards?parameters',
    automation: {
      trigger: {
        active: true,
        event: 'cardMoved',
        listName: 'In Progress'
      },
      labels: {
        display: true,
        color: 'brand-on-background-weak',
        tag: 'Data Structuring'
      },
      checklist: {
        display: true,
        items: 5,
        status: 'pending'
      },
      tracking: {
        display: false,
        color: 'neutral-alpha-weak',
        opacity: 100
      },
      dueDates: {
        display: true,
        color: 'accent-solid-strong',
        reminder: '24h'
      }
  }
}`, 
    language: "tsx",
    label: "scripts/trello_config.js"
  }
]} />
